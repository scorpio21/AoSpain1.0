Preguntas frecuentes
1. ¿Que es LiteStatsServer?
Es un servidor web diseñado especificamente para brindar servicios de mini estadisticás, recuperación de contraseñas y borrado de personajes.
2. ¿Que nescesita para funcionar?
Nescesita las Visual Basic Runtimes y los archivos ubicados en la carpeta "Nescesarios". También nescesita un servidor STMP para enviar los correos salientes.
3. ¿Cómo uso el programa?
Simplemente hay que ejecutarlo y comletar los datos que requiere.
Es importante completar los datos del server STMP ;).
También es nescesario que un FireWall no bloquee el puerto 80 y que no este siendo utilizado.
4. ¿Es seguro?
Eso espero :)... Es nescesario testearlo un poco, recordar que esta es la versión 0.1.
5. ¿Que pasa si quiero agregarle secciones?
El código esta disponible... ¡No es tan dificíl Visual Basic :)!
6. ¿Que ventajas tiene sobre Apache IIS?
No es tan pesado, no nescesita configuración, es más facil de configurar. También se puede citar el asunto de la seguridad. IIS y Apache tienen funciones que representan riesgos y que usted no usa ¿¿¿Porque correrlos???
